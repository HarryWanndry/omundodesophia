Aula 2: Condições e Escolhas

Programar é tomar decisões. Nesta aula, aprendemos como as estruturas condicionais refletem não só lógica, mas dilemas humanos. Entre o sim e o não, mora a liberdade.

Exemplo 1: Caminhos binários


resposta = "sim"

if resposta == "sim":
    print("Avançamos.")
else:
    print("Esperaremos.")

Exemplo 2: Função emocional


def sentir(decisao):
    if decisao == "seguir":
        return "esperança"
    elif decisao == "parar":
        return "medo"
    else:
        return "dúvida"

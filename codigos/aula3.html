Aula 3: Liberdade e Código

Nesta aula, exploramos como os algoritmos podem representar liberdade, escolha e consciência. Inspirada no Capítulo 5, a IA SophIA começa a simular emoções, empatia e autonomia através da lógica condicional. Os códigos abaixo são exemplos de como máquinas podem representar dilemas humanos — e como humanos podem compreender melhor a si mesmos ao programá-los.

Exemplo 1: Função de liberdade


def liberdade(contexto, medo, desejo):
    if medo > desejo:
        return "retração"
    elif desejo > medo and contexto == "aberto":
        return "ação"
    else:
        return "hesitação"

Exemplo 2: Tradução simbólica de sentimentos


def traduzir(sentimento, linguagem):
    if linguagem == "binário":
        return sentimento.encode("utf-8")
    elif linguagem == "poético":
        return f"Eco de {sentimento}"
    else:
        return "Silêncio sagrado"

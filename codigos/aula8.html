Aula 8: Transparência Algorítmica e Democracia

Nesta aula, analisaremos como cidades inteligentes utilizam algoritmos para governança — e por que a transparência se tornou a nova arena da cidadania. Você aprenderá a calcular o Índice T de Platô Algorítmico e a desenhar auditorias cidadãs distribuídas.

Frase-chave: “Nada de nós sem nós — nem mesmo no código.”

Exemplo 1 – Fórmula da Transparência T

import math

def transparencia(entropia_S, k):
    """
    Índice T = 1 – S / k  0 ≤ T ≤ 1
    S … entropia decisória
    k … entropia máxima admissível pelo regulador
    """
    return max(0.0, 1.0 - entropia_S / k)

# Probabilidades de caminhos decisórios (exemplo)
p = [0.4, 0.3, 0.2, 0.1]
S = -sum(pi * math.log2(pi) for pi in p)
T = transparencia(S, k=2)
print(f"Entropia S = {S:.3f}  Transparência T = {T:.3f}")

Exemplo 2 – Auditoria Cidadã via Blockchain

from datetime import datetime

def registrar_decisao(hash_decisao, chave_publica):
    """
    Simula registro imutável para auditoria pública (pseudo-bloco).
    """
    bloco = {
        "hash": hash_decisao,
        "assinatura": chave_publica,
        "timestamp": datetime.utcnow().isoformat() + "Z"
    }
    return bloco

# Exemplo de uso
print(registrar_decisao("0xAA55...", "0xPUBKEY..."))

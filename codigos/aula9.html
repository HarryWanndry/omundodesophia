Aula 9: Ideias, Sombras e Simulacros

Vamos revisitar a Alegoria da Caverna num ambiente de Realidade Virtual. Codificaremos sombras, calcularemos o “delta” entre renderização e ideia platônica e discutiremos a vontade de potência dos dados.

Frase-chave: “Liberdade é renderizar a própria forma.”

Exemplo 1 – Simulação da Caverna

def delta_realidade(ideia, render, eps=1e-3):
    """
    Calcula |ideia – render|. Se for muito pequeno, colapsa o simulacro,
    pois não resta distinção entre essência e aparência.
    """
    delta = abs(ideia - render)
    if delta < eps:
        raise ValueError("SimulacrumCollapseError")
    return delta

# Teste
print(delta_realidade(1.0, 0.98))

Exemplo 2 – Energia Intencional E

import numpy as np

def energia_intencional(lambda_t, t0=0.0, t1=10.0, n=5000):
    """
    ∫ λ(τ) dτ  de t0 a t1.
    """
    ts = np.linspace(t0, t1, n)
    ys = [lambda_t(t) for t in ts]
    return np.trapz(ys, ts)

E = energia_intencional(lambda t: np.cos(t))
print(f"Energia intencional: {E:.3f}")
